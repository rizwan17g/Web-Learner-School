Introduction to Typography in CSS

Typography is one of the most important aspects of web design. It affects readability, user experience, and the overall visual appeal of your website. CSS provides extensive control over font properties, text styling, and layout to help you create beautiful, readable content.

Good typography establishes hierarchy, guides the reader's eye, and conveys your brand's personality. Understanding font properties and text styling is essential for creating professional, accessible websites that communicate effectively with your audience.

CSS font and text properties work together to control every aspect of how text appears on your web pages, from the typeface and size to spacing, alignment, and decorative effects.

Key Principle: Typography should enhance readability and communication, not hinder it. Every typographic choice should serve your content and your users.

Font Family

The font-family property specifies which font to use for text. It's important to provide fallback fonts in case your preferred font isn't available on the user's system.

Font Family Examples
/* Single font family */
.serif {
    font-family: Times;
}

/* Font stack with fallbacks */
.modern-serif {
    font-family: "Georgia", "Times New Roman", Times, serif;
}

.sans-serif {
    font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;
}

.monospace {
    font-family: "Courier New", Courier, monospace;
}

/* Web fonts */
.custom-font {
    font-family: "Roboto", Arial, sans-serif;
}

/* System font stack */
.system-ui {
    font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
}

/* Generic font families */
.fantasy {
    font-family: fantasy;
}

.cursive {
    font-family: cursive;
}
Font Family Showcase
Serif Font (Georgia): Traditional and formal, great for body text and print-style layouts.
Sans-serif Font (Helvetica): Clean and modern, excellent for digital interfaces and headings.
Monospace Font (Courier): Fixed-width characters, perfect for code and technical content.
System Font: Uses the operating system's default interface font for familiarity.
Best Practice: Always include generic font families (serif, sans-serif, monospace) at the end of your font stack to ensure text renders properly even if specific fonts fail to load.

Font Size

The font-size property controls the size of text. Understanding different units and their use cases is crucial for responsive typography.

Font Size Examples
/* Absolute units */
.pixels {
    font-size: 16px;
}

.points {
    font-size: 12pt; /* Mainly for print */
}

/* Relative units */
.em-size {
    font-size: 1.2em; /* Relative to parent font size */
}

.rem-size {
    font-size: 1.5rem; /* Relative to root font size */
}

.percentage {
    font-size: 120%; /* Relative to parent font size */
}

/* Viewport units */
.viewport-width {
    font-size: 4vw; /* 4% of viewport width */
}

/* Keyword sizes */
.small {
    font-size: small;
}

.medium {
    font-size: medium;
}

.large {
    font-size: large;
}

.x-large {
    font-size: x-large;
}

/* Relative keywords */
.smaller {
    font-size: smaller;
}

.larger {
    font-size: larger;
}
Font Size Comparison
12px: Small text, fine print, captions
14px: Small body text, secondary information
16px: Standard body text size (browser default)
18px: Large body text, easier reading
24px: Subheadings, important text
32px: Section headings
48px: Page titles, hero text

Font Style

The font-style property controls whether text is displayed in normal, italic, or oblique style.

Font Style Examples
/* Normal text (default) */
.normal {
    font-style: normal;
}

/* Italic text */
.italic {
    font-style: italic;
}

/* Oblique text (slanted) */
.oblique {
    font-style: oblique;
}

/* Common use cases */
.emphasis {
    font-style: italic;
    color: #666;
}

.quote {
    font-style: italic;
    font-size: 1.1em;
    color: #555;
}

.technical-term {
    font-style: italic;
    font-weight: 500;
}
Font Style Examples
Normal text: This is the default, upright text style.
Italic text: This text is slanted and typically used for emphasis, quotes, or foreign words.
Oblique text: This is mechanically slanted text, similar to italic but computed rather than designed.

Text Alignment

The text-align property controls the horizontal alignment of text within its container.

Text Alignment Examples
/* Basic alignment */
.left {
    text-align: left;       /* Default for LTR languages */
}

.center {
    text-align: center;
}

.right {
    text-align: right;
}

.justify {
    text-align: justify;    /* Stretches text to fill line */
}

/* Responsive alignment */
.responsive-center {
    text-align: center;
}

@media (max-width: 768px) {
    .responsive-center {
        text-align: left;
    }
}

/* Common use cases */
.page-title {
    text-align: center;
    margin-bottom: 2rem;
}

.article-body {
    text-align: left;
    max-width: 65ch;        /* Optimal line length */
}

.button-container {
    text-align: center;
}
Text Alignment Demo
Left aligned (default): Most body text uses left alignment for easy reading and natural flow.
Center aligned: Great for headings, titles, and short important messages.
Right aligned: Less common, used for special effects or specific design needs.
Justified: This longer text demonstrates justified alignment, where text is stretched to fill the entire width of the container, creating straight edges on both sides. This can sometimes create awkward spacing between words.

Text Decoration

The text-decoration property adds decorative effects to text, such as underlines, overlines, and strike-through effects.

Text Decoration Examples
/* Basic decorations */
.underline {
    text-decoration: underline;
}

.overline {
    text-decoration: overline;
}

.line-through {
    text-decoration: line-through;
}

.no-decoration {
    text-decoration: none;      /* Remove default link underlines */
}

/* Modern text-decoration shorthand */
.fancy-underline {
    text-decoration: underline solid red 2px;
    /* line style color thickness */
}

.wavy-underline {
    text-decoration: underline wavy blue;
}

.dotted-overline {
    text-decoration: overline dotted green;
}

/* Individual properties */
.custom-decoration {
    text-decoration-line: underline;
    text-decoration-color: #3498db;
    text-decoration-style: dashed;
    text-decoration-thickness: 2px;
}

/* Hover effects */
.hover-underline {
    text-decoration: none;
    transition: text-decoration 0.3s ease;
}

.hover-underline:hover {
    text-decoration: underline;
}
Text Decoration Styles
Underlined text: Classic emphasis and link styling
Overlined text: Line above text (less common)
Strike-through text: Shows deleted or invalid content
Custom underline: Thick red solid underline
Wavy underline: Decorative wavy blue underline
Dotted underline: Subtle dotted green underline

Line Height

The line-height property controls the space between lines of text, crucial for readability and visual rhythm.

Line Height Examples
/* Different line height values */
.tight-spacing {
    line-height: 1;         /* Same as font size */
}

.normal-spacing {
    line-height: 1.4;       /* Good for body text */
}

.loose-spacing {
    line-height: 1.8;       /* More breathing room */
}

.pixel-line-height {
    line-height: 24px;      /* Absolute value */
}

.percentage-line-height {
    line-height: 150%;      /* Percentage of font size */
}

/* Responsive line height */
.responsive-line-height {
    font-size: 16px;
    line-height: 1.5;
}

@media (max-width: 768px) {
    .responsive-line-height {
        line-height: 1.4;
    }
}

/* Context-specific line heights */
.heading {
    line-height: 1.2;       /* Tighter for headings */
}

.body-text {
    line-height: 1.6;       /* Comfortable for reading */
}

.button {
    line-height: 1;         /* Compact for buttons */
}
Line Height Comparison
Tight spacing (line-height: 1.0):

This paragraph has very tight line spacing. Notice how the lines are close together with minimal space between them. This can make text harder to read, especially for longer content.

Normal spacing (line-height: 1.4):

This paragraph has normal line spacing, which provides a good balance between compactness and readability. It's commonly used for body text in many designs.

Loose spacing (line-height: 1.8):

This paragraph has loose line spacing with more breathing room between lines. This can improve readability for some content but may make text feel less cohesive.

.normal-letters {
    letter-spacing: normal;     /* Default */
}

.loose-letters {
    letter-spacing: 2px;

.wide-letters {

/* Word spacing */
.tight-words {
    word-spacing: -2px;
}

.normal-words {
    word-spacing: normal;       /* Default */

.loose-words {

    letter-spacing: 1px;

Spacing Examples

Normal letter spacing: Default spacing between letters
Loose letter spacing (2px): More space between letters

Text Transform Examples

Advanced Typography

Modern CSS provides advanced properties for fine-tuning typography and creating sophisticated text effects.

Advanced Typography Properties

}

.multiple-shadows {

Typography Best Practices

Creating effective typography requires understanding both technical implementation and design principles:

Readability Guidelines

  • Choose appropriate font sizes: 16px minimum for body text on screens
  • Maintain good contrast: Ensure sufficient color contrast between text and background
  • Control line length: Keep lines between 45-75 characters for optimal readability
  • Use appropriate line height: 1.4-1.6 for body text, tighter for headings

Hierarchy and Structure

  • Create clear hierarchy: Use size, weight, and color to establish information priority
  • Be consistent: Use a consistent type scale throughout your design
  • Limit font families: Use 2-3 font families maximum per design
  • Consider pairing: Choose fonts that complement each other

Performance and Accessibility

  • Optimize font loading: Use font-display: swap for web fonts
  • Provide fallbacks: Always include system font fallbacks
  • Test across devices: Ensure text renders well on different screens
  • Support zoom: Use relative units that scale with user preferences
Golden Rule: Typography should be invisible to the reader - it should facilitate communication without drawing attention to itself. Great typography enhances content, never competes with it.
