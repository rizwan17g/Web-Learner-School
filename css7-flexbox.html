Introduction to CSS Flexbox

CSS Flexbox (Flexible Box Layout) is a powerful one-dimensional layout method that makes it easy to arrange items in rows or columns. Flexbox was designed to provide a more efficient way to lay out, distribute space, and align items in a container, even when their size is unknown or dynamic.

Before Flexbox, creating flexible layouts required complex CSS involving floats, positioning, and clearfix hacks. Flexbox simplifies these tasks by providing intuitive properties for controlling alignment, distribution, and ordering of elements.

Flexbox is ideal for component-level layout, navigation bars, form controls, and any situation where you need to align items along a single axis. It works perfectly alongside CSS Grid, which handles two-dimensional layouts.

Key Concept: Flexbox works with a flex container (parent) and flex items (children). The container defines how items are arranged, while individual items can override certain behaviors.

Flex Container Basics

To start using Flexbox, you need to create a flex container by applying display: flex to a parent element. This immediately changes how the child elements are laid out.

Creating a Flex Container
/* Basic flex container */
.container {
    display: flex;
}

/* Inline flex container */
.inline-container {
    display: inline-flex;
}

/* Flex container with some styling */
.flex-container {
    display: flex;
    padding: 20px;
    background-color: #f8f9fa;
    border-radius: 8px;
}

/* Flex items (children) */
.flex-item {
    background-color: #3498db;
    color: white;
    padding: 20px;
    margin: 5px;
    border-radius: 4px;
    text-align: center;
}

Flex Direction

The flex-direction property defines the main axis and the direction flex items are placed in the container.

Flex Direction Options
/* Row direction (default) */
.flex-row {
    display: flex;
    flex-direction: row;        /* Items arranged horizontally, left to right */
}

/* Row reverse */
.flex-row-reverse {
    display: flex;
    flex-direction: row-reverse; /* Items arranged horizontally, right to left */
}

/* Column direction */
.flex-column {
    display: flex;
    flex-direction: column;     /* Items arranged vertically, top to bottom */
}

/* Column reverse */
.flex-column-reverse {
    display: flex;
    flex-direction: column-reverse; /* Items arranged vertically, bottom to top */
}

/* Responsive flex direction */
.responsive-flex {
    display: flex;
    flex-direction: row;
}

@media (max-width: 768px) {
    .responsive-flex {
        flex-direction: column;
    }
}

Justify Content

The justify-content property controls how flex items are positioned along the main axis (horizontal in row direction, vertical in column direction).

Justify Content Options
/* Align items to start (default) */
.justify-start {
    display: flex;
    justify-content: flex-start;
}

/* Align items to end */
.justify-end {
    display: flex;
    justify-content: flex-end;
}

/* Center items */
.justify-center {
    display: flex;
    justify-content: center;
}

/* Distribute space between items */
.justify-between {
    display: flex;
    justify-content: space-between;
}

/* Distribute space around items */
.justify-around {
    display: flex;
    justify-content: space-around;
}

/* Distribute space evenly */
.justify-evenly {
    display: flex;
    justify-content: space-evenly;
}

Align Items

The align-items property controls how flex items are positioned along the cross axis (vertical in row direction, horizontal in column direction).

Align Items Options
/* Stretch items to fill container (default) */
.align-stretch {
    display: flex;
    align-items: stretch;
}

/* Align items to start of cross axis */
.align-start {
    display: flex;
    align-items: flex-start;
}

/* Align items to end of cross axis */
.align-end {
    display: flex;
    align-items: flex-end;
}

/* Center items on cross axis */
.align-center {
    display: flex;
    align-items: center;
}

/* Align items to baseline */
.align-baseline {
    display: flex;
    align-items: baseline;
}

/* Perfect centering (both axes) */
.perfect-center {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 200px;
}

Flex Item Properties

Individual flex items can control their own behavior using flex properties: flex-grow, flex-shrink, and flex-basis, often combined into the shorthand flex property.

Flex Item Properties
/* Flex shorthand: grow shrink basis */
.flex-item-1 {
    flex: 1;                    /* flex: 1 1 0% - equal growth */
}

.flex-item-2 {
    flex: 2;                    /* Takes twice as much space as flex: 1 */
}

.flex-none {
    flex: none;                 /* flex: 0 0 auto - no growing or shrinking */
}

/* Individual properties */
.grow-item {
    flex-grow: 1;               /* Can grow to fill space */
    flex-shrink: 1;             /* Can shrink if needed */
    flex-basis: 0%;             /* Starting size before growing */
}

.fixed-width {
    flex: 0 0 200px;           /* Fixed 200px width, no grow/shrink */
}

.auto-width {
    flex: 0 1 auto;            /* Natural width, can shrink */
}

/* Align individual item differently */
.special-align {
    align-self: flex-end;       /* Override container's align-items */
}

Flex Wrap

By default, flex items try to fit on a single line. The flex-wrap property controls whether items wrap to new lines when there isn't enough space.

Flex Wrap Options
/* No wrapping (default) */
.no-wrap {
    display: flex;
    flex-wrap: nowrap;
}

/* Allow wrapping */
.wrap {
    display: flex;
    flex-wrap: wrap;
}

/* Wrap in reverse order */
.wrap-reverse {
    display: flex;
    flex-wrap: wrap-reverse;
}

/* Shorthand: flex-flow combines direction and wrap */
.flex-flow-example {
    display: flex;
    flex-flow: row wrap;        /* Same as flex-direction: row; flex-wrap: wrap; */
}

/* Responsive card layout */
.card-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 0 1 300px;           /* Minimum 300px, can shrink */
    min-width: 250px;          /* But not below 250px */
}

Flexbox Best Practices

Follow these guidelines to use Flexbox effectively and create maintainable layouts:

When to Use Flexbox

  • One-dimensional layouts: When arranging items in a single row or column
  • Component-level design: Navigation bars, button groups, form controls
  • Alignment challenges: Centering content, distributing space evenly
  • Flexible sizing: When items need to grow or shrink based on available space

Performance Considerations

  • Use with moderation: Don't nest flexbox containers unnecessarily
  • Prefer CSS Grid for 2D layouts: Grid is more efficient for complex layouts
  • Test on different devices: Ensure flexbox behaves as expected across browsers
  • Consider older browsers: Provide fallbacks if supporting legacy browsers

Common Mistakes to Avoid

  • Overusing flex: 1: Be specific about growth behavior when needed
  • Forgetting min-width: Flex items can shrink smaller than content
  • Not considering wrap: Plan for content overflow and wrapping behavior
  • Ignoring accessibility: Ensure keyboard navigation works with flex layouts
Golden Rule: Use Flexbox for component-level layouts and one-dimensional arrangements. Combine with CSS Grid for complete layout systems, and always test your layouts across different screen sizes and content lengths.
