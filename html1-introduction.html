What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of websites by using a system of tags to define different elements like headings, paragraphs, links, images, and more.

Think of HTML as the skeleton of a webpage. Just like how a skeleton gives structure to a body, HTML gives structure to a website. It tells the browser what content to display and how that content should be organized.

HTML was created by Tim Berners-Lee in 1990 and has evolved significantly over the years. The current version, HTML5, includes many new features that make web development more powerful and accessible.

Key Point: HTML is not a programming language - it's a markup language. It doesn't perform calculations or logic; it simply structures and presents content.

HTML Document Structure

Every HTML document follows a basic structure that includes several essential elements. This structure ensures that browsers can properly interpret and display your content.

Basic HTML Document Structure
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first paragraph of content.</p>
</body>
</html>

Understanding Each Part:

  • <!DOCTYPE html> - Declares the document type and version of HTML
  • <html> - The root element that contains all other elements
  • <head> - Contains metadata about the document (not visible on the page)
  • <title> - Sets the title shown in the browser tab
  • <body> - Contains all visible content of the webpage

HTML Tags and Elements

HTML uses tags to mark up content. Tags are keywords surrounded by angle brackets (< >). Most tags come in pairs: an opening tag and a closing tag. The closing tag includes a forward slash before the tag name.

An HTML element consists of the opening tag, the content, and the closing tag. Some elements, called void elements, don't have closing tags because they don't contain content.

Tag Examples
<!-- Opening and closing tags -->
<h1>This is a heading</h1>
<p>This is a paragraph</p>

<!-- Void elements (self-closing) -->
<br>
<img src="image.jpg" alt="Description">
<hr>
Remember: Always close your tags! Most HTML tags need both opening and closing tags to work properly.

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the opening tag and usually come in name/value pairs like name="value".

Attributes can control how an element behaves, what it looks like, or provide metadata about the element. Some attributes are required for certain elements to function properly.

Common Attributes
<!-- id attribute (unique identifier) -->
<div id="header">Header content</div>

<!-- class attribute (for styling) -->
<p class="highlight">Important text</p>

<!-- src attribute (image source) -->
<img src="photo.jpg" alt="A beautiful sunset">

<!-- href attribute (link destination) -->
<a href="https://www.example.com">Visit Example</a>

Why Learn HTML?

Understanding HTML is essential for anyone interested in web development. Here are some compelling reasons to master HTML:

  • Foundation of the Web: Every website uses HTML as its foundation
  • Easy to Learn: HTML has a simple, logical syntax that beginners can quickly grasp
  • Career Opportunities: HTML knowledge is required for web development jobs
  • Creative Expression: Build websites, blogs, and web applications
  • Problem Solving: Create solutions and tools for real-world problems
  • Always Relevant: HTML continues to evolve and remain essential
